# Commands
`ng` cmds will be available after angular cli is installed. 
<br/><br/>

### To create new angular project.
### `ng new <project-name>`
* project name convention:
  * lower case
  * `-` seperated.
### `npm start` or `ng serve`
* To start application.
<br/><br/>
